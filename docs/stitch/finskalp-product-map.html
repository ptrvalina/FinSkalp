# FinSkalp Product Map & Navigation Architecture ## I. Operational Center (Port 8877) 1. **Command Center** (`/`): High-level KPI dashboard, investigation funnel, and jurisdictional maps. 2. **OSINT Fusion Hub** (`/osint`): Investigation intake, Quick Score API triggers, and Scalpel collector management. 3. **Wallet Screening** (`/wallet`): Granular address analysis, 115-FZ regulatory recommendations, and flow mapping. 4. **Microservices Mesh** (`/microservices`): Topology of 24 core forensic services (Gateway, Risk Engine, Pattern Engine). 5. **Platform Modules** (`/platform`): Catalog of IC-01 to IC-08 intelligence components. 6. **Active Investigations** (`/ops`): Kanban/List view of cases with live telemetry and cross-chain graph integration. 7. **IC Console** (`/instruments`): Direct instrumentation access for transactional pattern monitors and corridor analyzers. 8. **Registries** (`/registries`): CIS VASP directory and Bank Hub connectivity status. 9. **Regulatory Reporting** (`/reports`): 115-FZ filing registry and PDF/HTML export center. 10. **System Telemetry** (`/status`): Uptime, collector health, and 'Combat Mode' status. ## II. Enterprise Web App (Port 5173) - **Compliance 115-FZ**: Deep lifecycle management, XGBoost risk scoring, and Evidence Graph (PostgreSQL/Neo4j). - **Flow Architect**: Visual pipeline editor for forensic automations. - **Vault**: Secret management and API key rotation. - **Investigation Workspace**: Multi-tab environment (Summary, Entities, Wallets, Timeline, Evidence, Graph).
