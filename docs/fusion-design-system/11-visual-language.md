# Visual Language

## Aesthetic

**Operational instrument** — not application UI.

## Surface Stack

| Layer | Color | Use |
|-------|-------|-----|
| Void | `#0a0c10` | Viewport background |
| Deck | `#0f1218` | Zone backgrounds |
| Panel | `#141820` | Panel chrome |
| Raised | `#1a1f28` | Hover, pinned |
| Divider | `#252b36` | 1px borders only |

## Shape

| Element | Radius |
|---------|--------|
| Panel | 2px (`--fusion-radius-sm`) |
| Button | 2px |
| Node | 4px |
| Rail icon hit | 4px |

No pills. No rounded-2xl.

## Elevation

Shadows forbidden except graph node pin glow (1px accent ring).

## Density

- Base font: 11px operational
- Strip labels: 9px uppercase
- Table rows: 28px
- Panel header: 32px
- Mission strip: 56px

## Iconography

Lucide stroke 1.5, 14px in rail, 12px inline. Opacity 0.85 — never decorative alone.

## Typography Pairing

IBM Plex Mono for data values; Inter for labels only.

## Rejected Visual Lineage

AdminLTE cards, Tailwind UI marketing blocks, shadcn card grids, gradient heroes, blob backgrounds.
